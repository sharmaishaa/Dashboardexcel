Vrinda Store Sales Analysis (Excel Dashboard Project)

**Project Overview**
This project focuses on analyzing Vrinda Store sales data for the year 2022 using Microsoft Excel.
The goal is to generate meaningful insights and help the business improve sales in 2023.

**Objective:**
Vrinda Store wants to:

Understand customer behavior
Identify top-performing regions and channels
Analyze sales patterns
Increase overall sales in the upcoming year
Project Contents
Raw Data (Excel)
Cleaned Dataset
Pivot Tables
Interactive Dashboard
 Insights & Conclusions

 Tools Used
Microsoft Excel
Pivot Tables
Charts (Bar, Pie, Line)
Slicers & Filters
Dashboard Design

**Key Analysis Questions**
-Compare sales and orders using a single chart
-Which month has the highest sales and orders?
-Who purchases more: Men or Women?
-What are the different order statuses?
-Top 10 states contributing to sales
-Relationship between age and gender
-Which sales channel contributes the most?
-Highest selling category


**Key Insights**
-Women are more likely to buy (~65%)
-Top 3 states: Maharashtra, Karnataka, Uttar Pradesh (~35%)
-Adult age group (30–49 yrs) contributes the most (~50%)
-Top channels: Amazon, Flipkart, Myntra (~80%)

**Final Conclusion**
To improve sales:
-Target women customers (30–49 yrs)
-Focus on Maharashtra, Karnataka, Uttar Pradesh
-Use ads, offers, and coupons
-Promote heavily on Amazon, Flipkart, Myntra
